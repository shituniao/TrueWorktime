## 概述
   工作计时器是一个帮助用户记录工作时长和时间管理的程序。   
   程序每秒检测当前正在使用的软件是否是预先设定的工作软件，且用户是否在30秒内有鼠标操作或键盘输入。

---

## 介绍
以及用户是否在30秒内有鼠标操作或键盘输入。
